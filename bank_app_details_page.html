Banking App: My first project

Ruby

I completed this project after my first few days at Coder Academy and 3 days of learning Ruby.

Week 1 at @CoderAcademy_au done!

Never thought I'd be able to make this a week ago but still a long journey ahead. pic.twitter.com/WmK4zzln13

The task was to implement as simple banking app. The features to include were:

  • Show the balance
  • Display a welcome message
  • Ask the user what they would like to do (e.g. check balance)
  • Error handle invlaid user input
  • Ask for a password before proceeding
  • Add deposit feature and update balance
  • Loop the program so user can deposit over and over
  • Clear the screen between loops
  • Add a withdraw feature and update balance
  • Prevent user from withdrawing more than their balance

Biggest challenges encountered

  • Giving the user 3 attempts at the password then denying access
  • Using Ruby methods for the first time
  • Bringing together mutliple features while keeping the application working

What I learned doing this project

  • Building a compelte application idea and putting together many features
  • First exposure to user input error handeling
  • sleep() function in Ruby
  • Continuous looping
